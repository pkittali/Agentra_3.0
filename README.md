# Agentra_3.0
Scalable, unified automation framework that supports:

✅ Web, Mobile, Desktop, and API testing
✅ Python + pytest stack
✅ Parallel runs
✅ Parameterization at all three levels
✅ Singleton pattern for drivers
✅ Abstracted driver layer (Selenium → Playwright switchable)
✅ Utilities for waits, data loading, logging, etc.
